## Datatypes:

## datatypes:

- int:
```cs
int a = 10;

Console.WriteLine(a.ToString());
```

- float:
```cs
float b = 53.005f;

Console.WriteLine(b.ToString());
```

- double:
```cs
double c = 2345.7652;

Console.WriteLine(c.ToString());
```

- bool:
```cs
bool d = true;

Console.WriteLine(d.ToString());
```
